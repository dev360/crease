Globex Bakeries sends a weekly delivery schedule. Each workbook tab is named for a week-ending date (e.g. 'W-04-25'). Within each tab are multiple per-day sections. Each section is bounded by a 'DELIVERY SCHEDULE' title in column D (start) and a long '=' separator row in column A (end — the LAST one before the next section, not the first). Between those, the section contains a date row, a projected-load summary to ignore, a column header row, and data rows grouped by depot with unlabeled subtotal rows between groups. The delivery date for every row is the date from that section's 'DAYNAME M-D-YYYY' row.
